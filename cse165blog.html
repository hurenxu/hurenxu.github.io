About The Project

The name of your project:
Fisherman's Diary

The names of the team members:
Renxu Hu
Yanqing Cheng

The short description of the project:
Our idea comes from the midterm, and we want to create exactly the same thing that comes from
the midterm, but with some creative modifications, it will be more interesting and fun to play.

Fisherman's Diary is an application that creates an immersive fish market, where
user can find an occean surrounded by a mountain terrain and a virtual fish market with the
symbolic input design.

In the occean, there will be some fishes that can be detected and captured by users.
After the capturing, the user will be able to sell the inventory to earn money.


The technical features you are implementing:
Selection: Go-Go
Symbolic Input: Virtual Keyboard
Manipulation: Go-Go with Button Inputs
System Control: Menu Design with action/toggle button, slider, choice list, and sub-menu
Wayfinding: minmap, and something else that we have not decided yet
Travel: not implemented yet
Spatial Audio: not implemented yet

What you are planning on spending your creative efforts on:
Implementing the fog style effects to give the illusion of being under water
Applying the manipulation with Go-Go to allow the capture of fish, the selling of fish
Integrated the spatial audio for every objects to create sound effects
Using the menu so that it allows the trade-making between user and computer
