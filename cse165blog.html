About The Project

The name of your project:
Fisherman's Diary

The names of the team members:
Renxu Hu
Yanqing Cheng

The short description of the project:
Our idea comes from the midterm, and we want to create exactly the same thing that comes from
the midterm, but with some creative modifications, it will be more interesting and fun to play.

Fisherman's Diary is an application that creates an immersive fish market, where
user can find an occean surrounded by a mountain terrain and a virtual fish market with the
symbolic input design.

In the occean, there will be some fishes that can be detected and captured by users.
After the capturing, the user will be able to sell the inventory to earn money. After
earning money, the user will be able to get higher-level tool to capture fish and can
capture the higher-level fish.

We will have two screens that attach both the user's left hand and the right hand. The
screen on the left will let the user know how many oxygen left, and the screen on the
right implements wayfinding techniques so that the user will know where the fishs are.


The technical features you are implementing:
Selection: Go-Go
Symbolic Input: Virtual Keyboard
Manipulation: Go-Go with Button Inputs
System Control: Menu Design with action/toggle button, slider, choice list, and sub-menu
Wayfinding: minmap and compass
Travel: Implemented both manual manipulation for mimicing driving the submarine
and implemented the viewpoint orientation techniques, such as head tracking
Spatial Audio: Different audio sources support all gameobjects and environment settings.

What you are planning on spending your creative efforts on:
We almost finish all the implementations, but there are still few more things left
First, at level 4, the tool is upgraded to a magic wand so the user will be able to
catch any fishes
Second, add a sound effect when the user catches the fish
Third, modified the fog, at different levels, the fog should be different
Fourth, add more fishes
